# humidor
Read temperature and humidity from my cigar humdor using Ruuvitag sensor. Data is read by Raspberry Pi and published in AWS via their IoT service.

## Dependencies

### Ruuvitag

To collect the data from my humidor.

https://tag.ruuvi.com
